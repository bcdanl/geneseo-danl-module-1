How many parties have provided positive funding contributions to other countries or regions for their adaptation projects for every single year from 2011 to 2018?

df = (
    climate_finance
    .query('`Type of support` == "adaptation"')
    .query('Contribution > 0')
    .groupby(['Party', 'Year'])

    .reset_index(name = 'n')

Q5

Calculate the Z-score of Close for each company on each Date \(t\).

# close_z is the right one.

df = (
    stock
    .assign(
        close_z =
         (stock['Close'] - stock_group['Close'].transform('mean'))
          / stock_group['Close'].transform('std'),
        close_z_all =
         (stock['Close'] - stock['Close'].mean())
          / stock['Close'].std(),

            )[['Date', 'company', 'Close', 'close_z', 'close_z_all']]
    .sort_values(['company', 'Date'], ascending = [True, False])

Q6

Use the transform() method on the stock data to represent all the values of Open, High, Low, Close, Adj Close, and Volume in terms of the first date in the data.

df = (
    stock
    .assign(
        open_rel = stock['Open'] / stock_group['Open'].transform('first'),
        high_rel = stock['High'] / stock_group['High'].transform('first'),
        low_rel = stock['Low'] / stock_group['Low'].transform('first'),
        close_rel = stock['Close'] / stock_group['Close'].transform('first'),
        adj_close_rel = stock['Adj Close'] / stock_group['Adj Close'].transform('first'),
        volume_rel = stock['Volume'] / stock_group['Volume'].transform('first'),
    )[['Date', 'company', 'open_rel', 'high_rel', 'low_rel',
       'close_rel', 'adj_close_rel', 'volume_rel']]
)
